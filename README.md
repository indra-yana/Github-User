# Github-User
Github user app - dicoding submission BFAA

## Features
- Kotlin
- MVVM Design Pattern
- Room Database
- Data Store Preferences 
- Retrofit
- Navigation Architecture Component
- Github User API ([GitHub REST API](https://docs.github.com/en/rest))
- etc
